# Omer Hayat Portfolio

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## Main content files

- `app/page.tsx`: homepage
- `app/projects/page.tsx`: projects
- `app/experience/page.tsx`: experience and certifications
- `app/writing/page.tsx`: newsletter and articles
- `app/notes/page.tsx`: reading list
- `app/components.tsx`: shared header and footer
- `app/globals.css`: site styling

The current résumé is available at `public/Omer_Hayat_Resume.pdf`.
